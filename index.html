Splitz App Case Study

Splitz is a travel assistance tool that allows travelers to take care of their group expenses all in one app. Record, split, charge, and pay your trip members in one app without wasting time going through multiple apps.

Problem Statement

We believe that offering an app to keep track of the expenses that occur while vacationing for those who travel in groups, will achieve a decrease in stress that is associated with finances during group travel.

User Testing

Data analysis from our User Testing constructed in a Feature Prioritization Matrix to help further understand what iterations are needed

Iterations made based off User Testing

The Apple sign-in feature should be down below the actual login button rather then directly below the email/password boxes. When signing up, there should be a verify account screen to connect the user directly to alleviate the user from having to re-login moving forward on the app. The homepage is very basic, it needs more life given to it as it is the main page every user encounters.
